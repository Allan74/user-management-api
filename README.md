
# 🛠️ User Management API

A simple **Express.js API** that allows users to **Create, Read, Update, and Delete (CRUD) users**.

## 🚀 Features
- ✅ Create a user  
- ✅ Fetch all users  
- ✅ Fetch a user by ID  
- ✅ Update a user  
- ✅ Delete a user  
- ✅ Organized using **Express Routers**  

## 🏗️ Tech Stack
- **Node.js**
- **Express.js**
